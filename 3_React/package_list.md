### 설치 방법
> <pre>
> npm install 패키지명
> yarn add 패키지명
> </pre>

#### yarn 사용 중 보안 에러 해결방법
* PowerShell을 관리자 권한으로 실행
* <code>set-ExecutionPolicy Unrestricted</code> 실행
  * 되돌리는 법 : set-ExecutionPolicy restricted
  * 보안정책 확인방법 : get-ExecutionPolicy
