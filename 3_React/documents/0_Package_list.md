## 설치 방법
> <pre>
> npm install 패키지명
> yarn add 패키지명
> </pre>

### Draft.js
* 텍스트 편집기 패키지
* React Draft Wysiwyg
  * Draft.js에 사용되는 라이브러리

### React Router Dom
* 설치
  > <pre>
  > npm install react-router-dom
  > yarn add react-router-dom
  > <pre>

### Node-sass
* 설치
  > <pre>
  > npm install node-sass
  > yarn add react-router-dom
  > </pre>
* import
  * css 파일에서
    > <pre>
    > @import "grid.scss"
    > @import "grid
    > </pre>
