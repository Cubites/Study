# 함수
## 함수 정의 방법
### 기본형
  > <pre>
  > function testFunc(){
  >     console.log("함수입니다.");
  > }
  > </pre>
### 함수 리터럴
  > <pre>
  > const testFunc = function(){
  >     console.log("함수입니다.");  
  > }
  > </pre>
### 익명 함수
  > <pre>
  > (function(){
  >     console.log("함수입니다.");  
  > })()
  > </pre>
  * 함수 생성과 동시에 실행됨
